Bench is an experiment around RDF storage performance and RDF exploration via SPARQL. It was built by four master students of the Université Montpellier 2 during a course on Semantic Web technologies and NoSQL databases. Bench tries to combine the best of both worlds and to experiment with as much data and as much storage solutions possible to compare them and make them stand out from each others.

This Web application is built in Java EE with the Apache Jena RDF framework. It also uses a custom built version of D2RQ and multiple databases, relational (PostgreSQL, MySQL, Databases As A Service) and NoSQL (Neo4J). Finally, the interface is built with Twitter Bootstrap, jQuery, CodeMirror and D3.js.
Our data is from INSEE, Nos Données.fr, PASSIM and GeoNames.

An exploration of RDF storage solutions. By Namrata Patel, Mathilde Salthun-Lassalle, Thibaut Marmin and Thibaud Colas.
